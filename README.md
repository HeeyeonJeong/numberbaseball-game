# Numberbaseball-game
numberbaseball game은 랜덤으로 만들어진 4자리 숫자를 맞추는 게임입니다. <br />
10번의 기회로 제한되면 숫자, 자리가 모두 맞으면 '스트라이크' , 숫자는 맞지만 자리가 다르면 '볼' 을 count하여 출력합니다.
## 📚 Stack
HTML, CSS, JavaScript
